O que é o modo web?
Entenda como funciona a versão web do Sistema de Etiquetas

Esta é uma versão de demonstração limitada do Sistema de Etiquetas que roda diretamente no navegador. Ela permite que você explore a interface e os recursos básicos sem precisar instalar o aplicativo completo.

No entanto, por limitações do navegador, algumas funcionalidades estão indisponíveis ou são apenas simuladas, como a impressão real de etiquetas e a comunicação direta com impressoras.

Recursos disponíveis
  • Visualização da interface completa
  • Navegação entre todas as telas
  • Gerenciamento de produtos (simulado)
  • Visualização de etiquetas de exemplo
  • Prévia de impressão
  • Configurações visuais
Recursos indisponíveis
  • Impressão real de etiquetas
  • Comunicação com impressoras físicas
  • Persistência de dados entre sessões
  • Atualizações automáticas
  • Recursos avançados de impressão
  • Alguns recursos específicos do desktop
